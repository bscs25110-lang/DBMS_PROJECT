About This Project

A database built for a Shopify store.

Stupify is a database management interface designed for a simplified Shopify-style e-commerce platform. It connects a structured MySQL backend to a clean HTML/CSS front end via a Flask server.

The schema covers everything from customers and orders to payments, shipping, reviews, and product categories — all visible in the ERD to the left.
