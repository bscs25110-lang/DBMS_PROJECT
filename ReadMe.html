Overview

Stupify is a database management web application designed as the front end for a simplified Shopify-style e-commerce store. It provides a clean interface to view, search, and manage product records stored in a MySQL database, served locally through a Python Flask backend.

The project was built as part of a university database design assignment. All queries run live against a relational database structured around a realistic retail schema.

Database Schema

The database consists of 9 tables modelling a complete e-commerce system:

  • CUSTOMER — customer profiles (name, phone, email)
  • ADDRESS — customer shipping addresses
  • POSTALCODE — city/country lookup by postal code
  • PRODUCT — products with price, stock, and category
  • CATEGORY — product categories
  • CART & CART ITEM — shopping cart sessions
  • ORDER & ORDER ITEM — placed orders and line items
  • PAYMENT — payment records linked to orders
  • SHIPPING METHOD — available shipping options
  • REVIEW — customer reviews on products

How to Run

Make sure you have Python and MySQL installed, then follow these steps:

# 1. Install dependencies
pip install flask mysql-connector-python

# 2. Set your DB credentials in app.py
#    host, user, password, database

# 3. Place HTML files in /templates
#    Place style.css and ERD Diagram.png in /static

# 4. Start the server
python app.py

# 5. Open your browser
#    http://localhost:5000

Tech Stack

  • HTML5 & CSS3 — front end structure and styling
  • Python 3 with Flask — server-side routing and templating
  • Jinja2 — template engine for dynamic data rendering
  • MySQL with mysql-connector-python — database layer
